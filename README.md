# ☁ 云课 - 智能课程表

一款简洁优雅的手机课程表应用，支持周视图展示、课程管理、自定义时间、暗色模式等功能。

---

## 功能特性

- **周视图课表**：以周一到周日的表格形式直观展示课程安排
- **课程管理**：添加、编辑、删除课程，支持课程名称/地点/教师历史记录快速选择
- **上课周次**：支持选择 1-25 周，可一键选择 1-16 周或 1-18 周
- **自定义时间**：可自定义每节课的上课时间，适配不同学校的作息安排
- **学期设置**：设置学期开始日期，自动计算当前周次
- **暗色模式**：支持亮色/暗色主题切换，自动跟随系统
- **手势操作**：左右滑动切换周次
- **数据本地存储**：所有数据保存在本地，无需联网，保护隐私

## 安装

### 直接安装 APK

将 `app-debug.apk` 或 `app-release.apk` 传输到 Android 手机，点击安装即可。

> 首次安装可能需要允许「安装未知来源应用」权限。

### 系统要求

- Android 5.1（API 22）及以上

## 使用指南

### 首次使用

1. 打开应用，进入周视图课表页面
2. 点击右上角齿轮图标进入设置页面
3. 设置**学期开始日期**（第一周周一的日期），以便正确计算当前周次
4. 如有需要，自定义每节课的上课时间

### 添加课程

有多种方式添加课程：

- **方式一**：点击底部导航栏中间的 **+** 按钮
- **方式二**：在周视图课表中，点击空白格子的 **+** 号（会自动填入对应的星期和节次）
- **方式三**：在课程列表页点击右上角 **+** 按钮

### 填写课程信息

添加课程时需填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 课程名称 | 必填，课程的名字 | 高等数学 |
| 上课地点 | 选填，教室位置 | 教学楼A-301 |
| 任课教师 | 选填，教师姓名 | 张教授 |
| 上课星期 | 选择周一到周日 | 周一 |
| 上课节次 | 选择开始和结束节次 | 第1节 — 第2节 |
| 上课周次 | 选择哪些周有课（必选） | 1-16周 |
| 课程颜色 | 选择课程卡片颜色 | 天蓝 |

> **历史记录**：输入课程名称、地点、教师时，如果之前输入过，右侧会出现下拉按钮，可以快速选择历史记录。

### 查看课程

#### 周视图

主页面为周视图，以表格形式展示一周所有课程：

- 表头显示星期和日期，当天列高亮标记
- 左侧显示节次和时间
- 课程以彩色卡片形式显示在对应位置
- 左右滑动可切换周次
- 点击课程卡片可编辑

#### 课程列表

点击底部导航栏「列表」按钮，进入课程列表视图：

- 顶部可切换查看不同星期的课程
- 列表显示课程的时间、名称、地点和教师
- 点击课程可编辑，右侧按钮可删除

### 编辑/删除课程

- **编辑**：在周视图或课程列表中点击课程卡片
- **删除**：在编辑页面底部点击「删除课程」按钮，或在课程列表中点击右侧删除图标

### 设置

点击周视图右上角齿轮图标进入设置页面：

- **学期开始日期**：设置第一周周一的日期，影响周次自动计算
- **上课时间**：自定义每节课的开始和结束时间
- **恢复默认**：将上课时间恢复为默认值

### 暗色模式

点击周视图右上角月亮/太阳图标，切换暗色/亮色模式。应用也会自动跟随系统暗色模式设置。

## 数据存储

所有数据使用浏览器 localStorage 存储，保存在本地设备中：

- 课程数据
- 输入历史记录
- 自定义上课时间
- 学期开始日期
- 暗色模式偏好

> ⚠ 卸载应用会清除所有数据，请谨慎操作。
